# Persona Journeys & Experience Rhythms

## 1. Momentum-Seeking Creator – “Jax”

- **Context**: Freelance illustrator juggling commissions. Morning dopamine crash after binge-stretch night.  
- **Trigger phrase**: “I have to sketch five panels but my brain won’t launch.”  

**Journey**
1. Jax opens Navi, types the trigger phrase.
2. Clarity AI flags low sentiment and “can’t start” language → Adaptive Engine predicts `LowDopamine`.
3. Navi responds with playful hype: “Let’s see how fast you can sketch one rough shape in 5 minutes.”
4. Task Translator presents `spark_challenge` micro-action + optional background beat.
5. Jax clicks “Done” after a few minutes. Momentum score increases; Navi transitions to `Regulated` and prompts a celebratory note.
6. Session saved with tags: `emotion=hopeful`, `project=comic panels`. Memory surfaces later as “wins you reclaimed.”

**Emotional Markers**
- Needs validation that difficulty = brain chemistry, not laziness.
- Responds to time-boxed games and visual cues.

## 2. Executive Function Juggler – “Mara”

- **Context**: Queer neurodivergent parent managing remote job + caregiver duties.  
- **Trigger phrase**: “My house is chaos, Slack is pinging nonstop, and my heart is racing.”  

**Journey**
1. Navi detects keywords “chaos,” “pinging nonstop,” sentiment < -0.4 → mode `Overwhelm`.
2. Tone shifts to soothing directive: “You’re carrying a lot. Let’s steady together.”
3. Micro-actions: `grounding_reset` (sensory grounding) and “Choose one thread: work or home? Navi will hold the other.”
4. Scheduler sets a 10-minute check-in. Anchors module plays calming chime.
5. Mara marks “Too much” on an action → engine logs overwhelm feedback, reduces intensity, offers “Quiet summary mode.”
6. Later, scheduler reminder surfaces with gentle question: “Is now a good moment for a hydration and inbox triage?”

**Emotional Markers**
- Safety-first language, permission to pause.
- Prefers defaults to quiet; audio optional.

## 3. Hyperfocus Guardian – “Eli”

- **Context**: Software engineer deep in code for 6 hours, forgetting meals.  
- **Trigger phrase**: “Still going, can’t stop now, just one more function.”  

**Journey**
1. Clarity pipeline notes neutral sentiment but long text, keywords “still going,” time gap 4 hours → `Hyperfocus`.
2. Navi replies with protective tone: “Your brain is sprinting hard. Let’s checkpoint to keep the sprint safe.”
3. Micro-actions include `checkpoint` (hydration + body scan) and “Jot quick plan for next commit before pausing.”
4. If Eli ignores, scheduler pushes reminder after 5 minutes with soft chime.
5. Once action completed, Navi acknowledges progress and suggests short restorative break before re-entering focus.

**Emotional Markers**
- Responds to logical framing (“protect your build”) plus subtle care.
- Avoids guilt; focus on preserving flow safely.

## 4. Post-Crash Rebuilder – “Sage”

- **Context**: After delivering project, Sage crashes emotionally, feels shame spiral.  
- **Trigger phrase**: “Project shipped, I’m empty and useless now.”  

**Journey**
1. Distortion detector flags catastrophizing + negative sentiment < -0.6 → `Crash`.
2. Navi mirrors emotion gently: “You’ve poured a lot out. The emptiness is signal, not failure.”
3. Micro-actions: `restoration` (self-compassion ritual) and journaling prompt (“What did your effort give someone?”).
4. Memories recall past recovery win: “Last time this feeling faded after a walk with music.”
5. After action, Navi logs mood shift, reminds Sage to schedule joyful activity tomorrow.

**Emotional Markers**
- Needs validation of worth independent of output.
- Appreciates memory callbacks (“Remember how you bounced back last month?”).

## 5. Therapist/Coach Companion – “Dr. Lian”

- **Context**: Works with clients on emotional regulation; wants between-session support without surveillance.  
- **Trigger use**: Reviews weekly summary exported by client; suggests custom prompts.

**Journey**
1. Client shares optional memory exports (CSV/Markdown).  
2. Dr. Lian reviews aggregated mode timeline and sees frequent `Overwhelm` evenings.
3. Adds custom template “Nightly soften & stretch” into client’s Task Translator library.
4. Navi syncs template locally; next evening, mode `Overwhelm` triggers new gentle suggestion.
5. Client reports success; therapist sees future summary noting improved evening regulation.

**Emotional Markers**
- Needs privacy assurances; data stays on client device.
- Wants ability to inject personalized interventions that align with therapy goals.

## Mode-Switch Experience Map

| Phase | Signals | Navi Response | User Feeling Target |
|-------|---------|---------------|---------------------|
| Entry | Text, sentiment, pace | Identify mode + reflect emotion | “Navi sees me.” |
| Guidance | Micro-actions, tone, optional cues | Offer 1-2 doable steps | “I can try that.” |
| Follow-up | Scheduler ping or check-in | Reaffirm progress, adjust mode | “I did something that counts.” |
| Memory | Save context, highlight wins | Feed future encouragement | “I’m building a compassionate record.” |

## Accessibility Considerations

- Toggle for text-only, no audio.
- Colour-safe mode indicator palette.
- Speed control for follow-ups (extend to 20–30 min).
- Optional haptic/lighting integration for sensory preferences.
